# jokenpo
Java version: 21

Para executar, faça o clone do projeto em sua maquina:
```
git clone https://github.com/natanxds/jokenpo.git
```

Entao acesse o diretorio contendo a classe principal: 
```
cd jokenpo/jokenpo/src/main/java
```
Faça a compilação:
```
javac Regra.java
```

Execute o programa:
```
java Main
```
